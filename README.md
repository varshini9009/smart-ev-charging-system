# smart-ev-charging-system
A responsive web application designed to help EV users locate charging stations and pre-book charging slots efficiently.
##Features
-user login 
-EV charging slots booking 
-Google Maps integration
-Dynamic pricing display
-Responsive user interface
##Technologies used
-HTML
-CSS
-JavaScript
##Project status
Currently under development.
